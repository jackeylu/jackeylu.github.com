# 怎样成为一个全栈程序员(full stack Developer)？

from 知乎

@匿名用户 
> 一个真正的全栈工程师，目标只有一个：**创业**！

@李楠

> **随着今天的分层越来越清晰，平台和语言越来越有特点，更加全面的技术人员可以根据不同的语言搭建整个架构**。

> **`Full Stack Developer`在快速原型上也很有优势，因为省去了大量的管理和沟通成本。而且，这并非意味着一定在代码质量或者测试上有缩水**。

> **Troble Shooting方面 `Full Stack Developer` 同样优势巨大**。

`Full Stack Developer`并非杂而全。

@另一个匿名用户[^ano]

1. 知识结构体系化，更扎实一些
2. 技术在某个方向做深
3. 切入某个应用领域，多了解领域知识，解决领域问题
4. 千万别沦为民科！
